## 协作约定

1. 每项功能先明确验收条件，再开始编码。
2. 算法接口或 CSV 格式变化时，必须同步更新设计文档和测试数据。
3. 合并代码前至少通过对应单元测试；演示版本需通过全部 CTest。
4. 文档、截图和 PPT 使用同一版本号，避免答辩材料与程序不一致。
